# Technologies Used
  Node.js v20.19.2
  Angular 14
  mongodb 5
# Api Docs 
  http://localhost:3000/api-docs/#/

# First Step Clone
  Clone Repo
# Go inside server folder
  1. Run npm i to install dependencies
  2. I have included .env in repo for testing so you didn't to add it manually
  3. Run mongodb server using mongod
  4. Install Node 20.19.2
  5. Run seed script to add dumy admin user by running npm run seed
  6. Then run npm start to run server
