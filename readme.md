# Technologies Used
  Node.js v20.19.2
  Angular 14
  mongodb 5

# First Step Clone
  Clone Repo
# Go inside server folder
  Run npm i to install dependencies
  I have included .env in repo for testing so you didn't to add it manually
  Run mongodb server using mongod
  Install Node 20.19.2
  Run seed script to add dumy admin user by running npm run seed
  Then run npm start to run server

  Username - admin
  password - Admin@123
